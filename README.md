All configuration parameters of the scenario are placed at util.Parameter.
By default, there are 40 tags and 10 readers in a 3*3 area.
And the number of slots is 100.
The reader's recognition radius is 1 and interference radius is 1.5.

The main method is in Main in package batch_test.
By running the program, we can get all data.
The data is stored in file multiTask_data in package data.
